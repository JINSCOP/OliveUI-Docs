* 名称定义：

| --- | --- |
| 主细表主表区域 | area-main-common-edit |

* 备忘笔记

area-main-common-edit:   主细表主表区域{
	三列或者四列，整站切换，要么整站三列要么整站四列（需要考虑，左边菜单展开的时候三列，收拢的时候四列），也有特殊情况，少数某几个页面可能通过后期用户配置决定到底是3列还是4列
	主表区域可能会有纵表（是独立页签还是和编辑界面融合，暂时不定）
}

	行数不确定
	
}
